# 开源库到JitPack

记录我走过的那些弯路，首先将自己lib写好然后我们需要简单的配置一下上传到github。

# 1. 添加maven构建插件

Project:build.gradle

    repositories {
        mavenCentral()
    }
    dependencies {
        classpath 'com.github.dcendents:android-maven-gradle-plugin:2.1'
    }

你开源的Lib:build.gradle

apply plugin: 'com.github.dcendents.android-maven'
group='com.github.xxx' //xxx替换成你自己的Github用户名

# 2. 发布项目

  1. 上传项目至github
  2. Releases->添加Tag->添加描述

# 3.使用

  1. 登录JitPack搜索自己的项目 get it
  2. 按照How to配置即可

# 注意

有时候会遇到一些奇奇怪怪的问题，去重lib的依赖往往可以解决这些问题。

implementation('com.github.littonishir:hellomvp:1.0.3') {
        exclude group: 'com.android.support'
    }
