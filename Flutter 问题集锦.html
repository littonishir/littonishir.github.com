# Flutter 问题集锦

工具: mac电脑 android studio

# 1. Waiting for another flutter command to release the startup lock...

Flutter工程 连接不到设备

问题原因：

项目异常关闭，下次启动就会出现上面的一行话。

解决办法：

在Finder中搜索lockfile 找到文件删除即可。
