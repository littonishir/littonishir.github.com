# 一 注释

Dart的注释分为3种：单行注释、多行注释、文档注释。

 // 单行注释 superstars

 /**
  * 多行注释
  * We are the superstars, super, super, superstars
  * 我们是巨星
  * Yeah, we the ones, who have the power
  * Yeah 我们就是有权有势
  */

 /// 文档注释可以通过dartdoc命令导出文档 Trust me, I will give you honor

# 三 变量和常量

# 变量声明

var普通变量创建, Object /dynamic没有明确类型或对象不限于单一类型

 var age = 3;
 Object x = 'x';
 dynamic y = 'y';

默认值:未初始化的变量的初始值均为null（包括数字）

# 常量声明

final 变量只能被设置一次;const 变量是一个编译时常量。

 final see = "I won't leave until you see me";
 const steal = "I'm about to steal your high";
  1. 被final或者const修饰的变量，变量类型可以省略。
  2. flnal 或者 const 不能和 var 同时使用
  3. 常量如果是类级别的，请使用 static const

# 内置的类型

  • numbers
  • strings

  • lists (也被称之为 arrays)
  • maps
