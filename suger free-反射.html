# 反射机制

JAVA反射是在运行时,对于任意一个类,都能知道这个类的所有属性和方法;对于任意一个对象,都能够调用它的任意一个方法和属性;这种动态获取的信息以及动态调用对象的方法的功能称为Java语言的反射机制

# 反射使用

使用反射的前提:必须先拿到目标类的字节码(Class) Class其实也是一个对象，字节码是由 Constructor Field Method 构成

# 获取Class对象的方法


  1. 调用类的class属性
  2. 调用对象的getClass()方法
  3. 使用Class类中的forName()静态方法(最安全/性能最好)
        // 1.调用类的class属性
        Class<Games> gameClass1 = Games.class;
        // 2.调用对象的getClass()方法
        Games games = new Games("贪玩蓝月", "端游");
        Class<? extends Games> gameClass2 = games.getClass();
        // 3.使用Class类中的forName()静态方法(最安全/性能最好)
        try {
            gameClass3 = Class.forName("com.litton.ishirdaily.bean.Games");
        } catch (ClassNotFoundException e) {
            e.printStackTrace();
        }

# 访问构造(Constructor)


  • 获取 Constructor
    1. getConstructor 获取所有"公有的"构造方法.
    2. getDeclaredConstructor (获取所有的构造方法(包括私有/受保护/默认/公有)
  • 实例化对象
    1. getConstructor().newInstance()
    2. getDeclaredConstructor(parameter.class).newInstance(parameter)
  try {
        // 获取两个参数的构造方法 而且参数是String类型
        Constructor constructor = Games.class.getDeclaredConstructor(String.class，String.class);
        constructor.setAccessible(true);// 设置权限
        // "贪玩蓝月", "端游" 调用构造方法传的参数 == new Games("贪玩蓝月", "端游");
        Games game = (Games) constructor.newInstance("贪玩蓝月","端游");
        } catch (Exception e) {
            e.printStackTrace();
        }

# 访问属性(Field)


  1. 获取 Field: getField(String fieldName)
  2. 获取 Field 的值:
    • setAccessible(true)
    • field.get(Object obj)
  3. 设置 Field 的值: field.set(Obejct obj, Object val)
try {
            Field f = gameClass1.getDeclaredField("name");
            f.setAccessible(true);
            f.set(games1, "{影之诗}");
            Log.e(TAG, "Modify field: " + f.get(games1));

            Field[] fields = gameClass1.getDeclaredFields();
            for (Field field : fields) {
                field.setAccessible(true);
                Log.e(TAG, "field: " + field.get(games1));
            }

        } catch (Exception e) {
            e.printStackTrace();
        }

获取ActivityThread的属性

        try {

            Field sCurrentActivityThreadField =  clazz.getDeclaredField("sCurrentActivityThread");

# 访问方法(Method)


  1. 获取 Method:
    • getDeclaredMethods: 得到 Method 的数组.
    • getDeclaredMethod(String methondName, Class ... parameterTypes)
  2. 调用 Method
    • 方法被private修饰,需先调用 Method.setAccessible(true),使其可访问
    • method.invoke(obj, Object ... args);
try {
            Method method = gameClass1.getDeclaredMethod("setName",String.class);
            method.setAccessible(true);
            method.invoke(games1,"{Suger fee}");
            Log.e(TAG, "Game: " + games1.toString());

            Method[] methods = gameClass1.getMethods();
            for (Method method : methods) {
                String name = method.getName();
                Log.e(TAG, "Method List: " + name);
            }
        } catch (Exception e) {
            e.printStackTrace();
        }
